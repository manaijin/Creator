# 创世者

## 立意

&emsp;&emsp;人生于混沌，但皆有探索世界之欲，求毕生之志。张载说“为天地立心，为生民立命，为往圣继绝学，为万世开太平”，这是儒家的极致追求。司马迁欲“究天人之际，通古今之变，成一家之言”，将定史作为人生追求。佛曰“一切有为法，如梦幻泡影，如露亦如电”，对世间法则作出了自己的理解。

&emsp;&emsp;生而为创世者，大道朝天，各走一边。食百家饭，历千般劫，结一世果。
